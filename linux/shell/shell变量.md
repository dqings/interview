# shell 变量

## 特殊状态变量

| 状态变量 | 作用                                                       |
| -------- | ---------------------------------------------------------- |
| $?       | 获取上一个执行指令的执行状态返回值(0为成功,非0为失败) 常用 |
| $$       | 获取当前执行的shell脚本的进程号(PID)                       |
| $!       | 获取上一个在后台工作的进程号                               |
| $_       | 获取在此之前执行的命令或脚本的最后一个参数                 |
