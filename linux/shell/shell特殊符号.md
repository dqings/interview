# shell特殊符号

```shell
''  所见即所得，吃啥吐啥
""  与 '' 效果一样，里边的特殊符号会被解析运行
``  ===$() 先运行里边的命令，把结果留下
#######################
>   重定向符号，先清空里边的内容，然后追加到文件的最后
>>  追加重定向，追加到文件的最后，不清空
2>  错误重定向，只有错误的信息，才会通过这个漏洞进入文件中
2>> 错误追加重定向 不清空文件
#######################
~   当前用户的家目录
# root === /root
# dqings === /home/dqings
!   查找并运行历史命令
#   1、注释  2、root用户的命令提示符
$   普通用户的命令提示符
*   任意东西
\   转义字符
&&  前一个命令执行成功，在执行后边的命令
||  前一个命令执行失败了，在执行后边的命令
&   后台运行
```
